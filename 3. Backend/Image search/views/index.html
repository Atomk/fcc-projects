Image Search Abstraction Layer

This API lets you search for images and the view the latest searches.

To search images about pineapples:
https://atomk-fcc-imgsearch.glitch.me/search/pineapple

Example output:
[{ "link":"http://xxx.yyy/img.jpg", "snippet": "Pineapples are the best", "context": "http://xxx.yyy/pineapples.html" }, ... ]

To view the latest searches:
https://atomk-fcc-imgsearch.glitch.me/latest

Example output:
[{"term":"pineapple","time":"2018-01-06T10:21:31.642Z"}, {"term":"banana","time":"2018-01-06T10:24:06.131Z"}, ... ]
